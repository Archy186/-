# -# Задание 1: Настройка балансировки Round-robin на 4 уровне с помощью HAProxy

## 📋 Описание задания
В рамках данного задания выполнена следующая работа:
- Запущены два простых Python сервера на разных портах
- Установлен и настроен HAProxy
- Настроена балансировка Round-robin на 4 уровне (TCP)
- Проверено перенаправление запросов на разные серверы при обращении к HAProxy
